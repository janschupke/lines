Lines - User's Guide

Overview

Lines is a logical game based around arranging balls of the same color into lines. Each turn, a player can pick any one of the balls and move it to any free cell on the board, if this cell is reachable by a direct path. If this action forms a line of at least 5 balls of the same color, that line is removed from the board and the player is awarded a certain amount of points.

Multiple lines can be removed in 1 turn, if they meet the criteria stated in the above paragraph. If any line is removed from the board, the next turn starts without spawning any additional balls. However, if a ball is moved and this action does not create a sufficiently long line, three new balls are spawned at the beginning of the following turn.

Game ends either by restarting it through the menu, or by filling up the entire board with balls. If the player's score at the end of the game is high enough, he is given a choice to save this result into the Score Board. Player's name can be modified at that point.

High enough score either means that the Score Board is not fully filled up yet, or the currently reached score is higher than the lowest entry in the Score Board. Also, smaller board size is always prioritized when saving a score entry, regardless of the amount of points achieved (large amount of score points achieved by playing on larger board will not be saved, if the Score Board is filled up by entries achieved on smaller board).

Game Window

The game window consists of three main parts:

  1. Info Panel at the top,
  2. Game Board with Cells in the middle,
  3. Status Bar at the bottom.

Info Panel

Info Panel contains information about the player's name and his current score, and also shows what types of balls (colors) will be spawned the next time the spawning is needed (end of turn that did not involve line removing).

Player name in the top left corner of the window can be modified by clicking at the text. Pressing the Enter button confirms the name change, pressing the Escape button cancels any changes made. Name modification is also available during the score-saving process in a separate dialog.

Icons of incoming balls can be hidden by checking off the appropriate box in the configuration (F2).

Game Board

The main Game Board is a grid of cells that can be clicked to issue ball movement. No dragging is allowed. Left-clicking a ball makes it active, another left click on any reachable free cell moves the ball to that destination. Right click anywhere on the board cancels the current turn.

If the highlighing feature is enabled in the configuration, empty cells' background is changed every time a ball is activated, to indicate whether the given cell is reachable from that position.

Cell

Cells are the individual squares that the main board consists of. There are two types of ball images that can be painted onto a cell:

The smaller versions of balls represent hints about positions where these balls would spawn at the beginning of the next turn. These cells are still considered empty, cannot be activated and are viable destinations for any ball that can reach them by a direct path. If the player chooses to move a ball onto any position that expects a ball spawn, the actual spawn location for that particular incoming ball is recalculated when needed. [hint]
The full-sized images represent the actual balls that the player can activate and move. [ball]

Status Bar

Status Bar consists of two parts. Whenever the mouse cursor is moved over anything that has a displayable tooltip, this tooltip is, in conjunction with the floating bubble at the point of the mouse cursor, shown in the left part of the Status Bar. On the right-hand side, timers for the current turn and total game time are displayed.

Application Controls

The application is mostly mouse-driven. Gameplay is done entirely by left mouse clicks. However, some hotkeys have been added for convenience. Full hotkey list is displayed below:

List of Hotkeys

Hotkey Description
F1 Starts a new game
F2 Opens the Configuration
F3 Open the Score Board
F4 Opens the About tab
F5 Opens this Guide
Escape Deselects selected cell (ball)
N Starts new turn
<CTRL> + <ALT> + Q Exits the Application
